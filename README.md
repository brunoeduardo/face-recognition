# Face Recognition Project using Face-API

This project implements face detection and recognition using the face-api.js library. 

## Features detections
 - Face  
 - Recognition 
 - Landmarks 
 - Facial Expressions
 - Real-time Video 

## Technologies Used
 - face-api.js
 - JavaScript
 - HTML/CSS
 - Node.js(locally server)
